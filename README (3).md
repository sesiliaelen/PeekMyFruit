# PeekMyFruit 🍎🍌🍊

Aplikasi pendeteksi tingkat kematangan buah (**Mentah / Matang / Busuk**) dari
gambar, menggunakan MobileNetV2 (transfer learning).

Unggah gambar buah atau ambil foto langsung, dan model akan memperkirakan tingkat
kematangannya beserta rekomendasi waktu konsumsi.
